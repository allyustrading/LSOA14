# Liuli Society of America — Production-Ready Homepage

This project is a production-ready Vite + React homepage built to match the provided classic reference layout while using real similar stock photos placed as local assets.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Notes
- Images are stored in `src/assets`
- Styling is in `src/styles.css`
- The site is configured for static hosting with `base: './'`
